# 命令列表

Xepod 希望提供简单且直观的命令使用方式，其主要命令如下。

## help

打印帮助列表

```
xepod help
```

## update

更新 rack 数据

```
xepod update
```

## list

查询已安装的 app ，如无查询字段，列出全部已安装 app

```
xepod list <query>
```

## search

查询 rack 中可安装的 app ，如无查询字段，列出全部可安装 app

```
xepod search <query>
```

## install

安装 app

```
xepod install [app]...
```

## remove

```
xepod remove [app]...
```

## upgrade

```
xepod upgrade <app>...
```
